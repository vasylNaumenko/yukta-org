Некоторые рекомендации

по использованию санскрито-англо-русского словаря.

Санскрито-англо-русский словарь содержит для каждого санскритского значения одно или более русских и английских значений. Таким образом, словарь содержит три раздела и Вы можете осуществлять поиск в каждом из этих разделов.

Задача

Решение

1

Найти в словаре санскритское значение

А

Используя поисковую машину. В разделе 'Найти' введите искомое санскритское слово* или его маску** для поиска, выберите область поиска 'в санскрите' (по умолчанию) и нажмите 'Искать'. В случае успешного поиска будет загружен список найденных значений, из которого Вы можете выбрать нужные для дальнейшего просмотра.

* В словаре используется транслитерация Харвард-Киото (см. Приложение 1). Примеры некоторых санскритских значений в разных кодировках приведены в сравнительной таблицы Приложения 2. Если Вы не знаете как точно пишется искомое Вами значение, попробуйте найти его, используя 'Список всех значений' (см. п. Б ниже) либо используя маску для поиска.

** В маске для поиска санскритских значений можно использовать нижнее подчеркивание '_' для обозначения одного произвольного символа и '%' или '*' - для обозначения любого количества символов, включая их отсутствие. Например, по маске _iva будет найдено diva, jIva, ..., а по маске %iva или *iva будет найдено atIva, daiva, diva, iva, jIva, tathaiva, ...

Б

Используя полный список санскритских значений. В разделе 'Список всех значений' выберите раздел списка, который может включвть в себя искомое слово, и щелкните по кнопке 'Показать' (сортировка списка – латынь, не Деванагари). Загрузится список значений выбранного раздела, Просмотрите его и выберите нужное, щелкнув по нему. Если нужны несколько значений – отметьте их и нажмите Enter.

2

Найти санскритские значения в переводе которых встречается указанное русское значение.

В разделе 'Найти' введите искомое русское слово или его маску (начало слова и символ '*' ,например, радос*), выберите область поиска 'в русском' и нажмите 'Искать'. В случае успешного поиска будет загружен список найденных значений, из которого Вы можете выбрать нужные для просмотра.

3

Найти санскритские значения в переводе которых встречается указанное ангийское значение.

В разделе 'Найти' введите искомое английское слово или его маску (начало слова и символ '*' ,например, worl*), выберите область поиска 'в английском' и нажмите 'Искать'. В случае успешного поиска будет загружен список найденных значений, из которого Вы можете выбрать нужные для просмотра.
